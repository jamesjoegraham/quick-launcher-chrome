# Quick Site Launcher

A Chrome extension that opens up to 4 user-defined URLs via keyboard shortcuts.

## Installation

1. Go to `chrome://extensions` and enable **Developer mode**.
2. Click **Load unpacked** and select this folder.

## Configuration

### Set your URLs

1. On `chrome://extensions`, find Quick Site Launcher and click **Details → Extension options**.
2. Enter a URL for each slot and click **Save Shortcuts**.

### Assign keyboard shortcuts

1. Go to `chrome://extensions/shortcuts`.
2. Find Quick Site Launcher and assign a key combination to each "Opens Custom Site" command (e.g. `Ctrl+Shift+1`).

## Usage

Press a configured shortcut from any Chrome tab to open the corresponding URL in a new tab.
